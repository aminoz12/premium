# Manual SEO Tasks - Action Required

This document outlines SEO tasks that require manual action by you or your team.

## 1. Google Search Console (GSC) Setup

### Steps to Complete:
1. **Create Google Search Console Account**
   - Go to https://search.google.com/search-console
   - Sign in with your Google account
   - Click "Add Property" and select "URL prefix"
   - Enter: `https://premiumiptv.live/`

2. **Verify Ownership**
   - Choose HTML file upload method
   - Upload the verification file to your public folder
   - Or use DNS verification (recommended for production)

3. **Submit Sitemaps**
   - Go to Sitemaps section in GSC
   - Submit: `sitemap_index.xml` (this includes all other sitemaps)
   - Submit: `sitemap_images.xml`
   - Wait for Google to index (usually 24-48 hours)

4. **Submit Individual Sitemaps**
   - `sitemap.xml` (main pages)
   - `sitemap_blog.xml` (when created)
   - `sitemap_video.xml` (when created)

5. **Set International Targeting**
   - Go to Settings → International Targeting
   - Set country targeting to "Unlisted" (since you target multiple countries)
   - Your hreflang tags will handle country-specific targeting

6. **Monitor Index Coverage**
   - Check Index Coverage report
   - Fix any errors or excluded pages
   - Ensure all important pages are indexed

7. **Check Mobile Usability**
   - Review Mobile Usability report
   - Fix any mobile issues
   - Ensure responsive design works perfectly

8. **Monitor Core Web Vitals**
   - Check Core Web Vitals report
   - Target: LCP < 2.5s, FID < 100ms, CLS < 0.1
   - Optimize images, reduce JavaScript, improve server response time

### Priority: HIGH
### Time Required: 30 minutes
### Frequency: Weekly monitoring
